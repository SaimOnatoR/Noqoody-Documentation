Website Documentation

Introduction

Noqoody is a Retina Ready Responsive HTML webasite. Developed with Bootstrap 5 and Sass.

Also, the website has RTL.

Noqoody.com is fast, lightweight, clean and easily customizable and expandable with details documentation.

Website Features:
  • Multipage
  • Easy to customize
  • HTML5 & CSS3
  • Clean & Simple Design
  • Fully Responsive Layout
  • W3 Valid
  • Retina ready
  • FontAwesome and Flaticon icons
  • Powered with Bootstrap (v5.x)
  • Powered with jQuery (v3.6.4)
  • Build with Sass
  • AJAX contact form Submission
  • AJAX subscriptions form Submission
  • Well documented
  • Crossbrowser Compatible with Edge, IE11+, Firefox, Opera, Chrome

Fonts used

By default, the template loads Roboto and Raleway font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.css" file:


@import url('https://fonts.googleapis.com/css?family=Roboto:100,100i,300,300i,400,400i,500,500i,700,700i,900,900i&display=swap');

How to Hide Scroll Animation

Please remove or comment the below code file path: assets/js/main.js





// WOW JS


$(window).on ('load', function (){


    if ($(".wow").length) {


        var wow = new WOW({


        boxClass:     'wow',      // animated element css class (default is wow)


        animateClass: 'animated', // animation css class (default is animated)


        offset:       20,          // distance to the element when triggering the animation (default is 0)


        mobile:       true, // trigger animations on mobile devices (default is true)


        live:         true,       // act on asynchronously loaded content (default is true)


      });


      wow.init();


    }


});
